# sccache in Base Image + Shared Compile Cache — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Bake [sccache](https://github.com/mozilla/sccache) into the shared base image as the Rust compiler cache and persist it on the host, shared across all agent containers.

**Architecture:** Install sccache in `Dockerfile` via `cargo binstall`, wire cargo through it with `RUSTC_WRAPPER`, pin the cache dir with `SCCACHE_DIR`. Each `run-*.sh` bind-mounts the host cache (`~/.cache/sccache`) into the fixed container path so every run reuses one shared cache.

**Tech Stack:** Docker (debian:trixie-slim base), Rust/cargo, cargo-binstall, sccache, bash launchers.

**Spec:** `docs/superpowers/specs/2026-06-06-sccache-design.md`

**Branch:** `feat/sccache` (already checked out; spec commit `7f95928` lives here).

**Note on "tests":** This is infra/config work — no unit tests. Verification = Docker build success, image sanity checks, and a runtime cache-hit check. Treat the verification steps as the test gates.

## Task 1: Add sccache to the base image

**Files:**
- Modify: `Dockerfile`

- [ ] **Step 1: Add the `SCCACHE_VERSION` build arg**

Find the existing `ARG RTK_TAG=v0.42.0` line (~L74). Immediately above the rtk
`ARG RTK_TAG` block, add:

```dockerfile
# sccache version for cargo-binstall. Bare semver (no leading "v"): binstall's
# version spec is `sccache@X.Y.Z`. Override with --build-arg SCCACHE_VERSION=X.Y.Z.
ARG SCCACHE_VERSION=0.8.2
```

- [ ] **Step 2: Add the sccache install RUN layer**

Insert this RUN **after** the rtk install RUN block (the one ending in
`rtk --version`, ~L88) and **before** the sanity-check RUN (~L91):

```dockerfile
# sccache: compiler cache for Rust. Installed with cargo-binstall (prebuilt
# binary, no source compile). cargo-binstall itself comes from its official
# prebuilt installer script. Both land in $CARGO_HOME/bin (already on PATH).
RUN set -eux; \
    curl -fsSL --proto '=https' --tlsv1.2 \
        https://raw.githubusercontent.com/cargo-bins/cargo-binstall/main/install-from-binstall-release.sh \
        | bash; \
    cargo binstall --no-confirm "sccache@${SCCACHE_VERSION}"; \
    sccache --version; \
    chmod -R a+rwX "$CARGO_HOME"; \
    rm -rf /root/.cache /tmp/*
```

- [ ] **Step 3: Add the Rust ENV vars**

Add a new ENV block immediately after the sccache RUN layer from Step 2:

```dockerfile
# Route all cargo builds through sccache, and pin the cache dir so the runner
# bind-mount target is deterministic even under rootless Docker (container runs
# as root, HOME=/root, so the default ~/.cache/sccache would miss the mount).
ENV RUSTC_WRAPPER=sccache \
    SCCACHE_DIR=/home/dev/.cache/sccache
```

- [ ] **Step 4: Add sccache to the sanity-check RUN**

Find the sanity RUN line (~L92):

```dockerfile
    rustc --version; cargo --version; clippy-driver --version; rust-analyzer --version; \
```

Append `sccache --version;` to it:

```dockerfile
    rustc --version; cargo --version; clippy-driver --version; rust-analyzer --version; sccache --version; \
```

- [ ] **Step 5: Build the base image (verification gate)**

Run: `docker build -t agentic-dev-base:latest .`
Expected: build succeeds; output shows a cargo-binstall fetch of sccache and an
`sccache X.Y.Z` version line from both the install layer and the sanity RUN.

- [ ] **Step 6: Verify the image wiring**

Run:
```bash
docker run --rm agentic-dev-base:latest \
  bash -lc 'echo "wrapper=$RUSTC_WRAPPER dir=$SCCACHE_DIR"; sccache --version'
```
Expected: prints `wrapper=sccache dir=/home/dev/.cache/sccache` and an
`sccache X.Y.Z` line.

## Task 2: Mount the cache in run-claude.sh

**Files:**
- Modify: `run-claude.sh`

- [ ] **Step 1: Add host cache dir resolution**

Find the `WORK_DIR="$PWD"` line (~L57). After the block of top-level var defaults
(after `MEM_TO=0`, ~L63), add:

```sh
# Persistent sccache compile cache, shared with the host. Pin to the host default
# (~/.cache/sccache) unless SCCACHE_DIR is already exported. Created before the
# mount so Docker doesn't materialize a root-owned dir.
SCCACHE_CACHE="${SCCACHE_DIR:-$HOME/.cache/sccache}"
mkdir -p "$SCCACHE_CACHE"
```
